    查询接口

    接口地址

    http://pay.congmingpay.com/pay/query.do

    参数说明

    字段名 变量名 必填 类型 示例 描述
    商户订单id orderId 三选一必填 String（32） Abcd12345 商户的订单id
    聪明付订单id cmOrderId String（32） 20160922194944712595 聪明付的订单id
    通道流水号 transaction_id String（32） 101550066030201708281221607314 通道流水号，退款单号
    商户id shopId String（32） c2e17899960c83c2aeca03c37b13ade0 聪明付的商户号
    退款单号 refund_order_id 否 String（32） Abc123456 商户的退款订单号，优先级最低

    响应内容

    响应内容为json格式字符串，字段如下：

    字段名 变量名 必填 类型 示例 描述
    支付状态 result_code 是 String（20） SUCCESS/FAIL 查询成功为SUCCESS，查询失败为FAIL
    失败结果 result_msg 否 String（64） 订单号不存在 查询失败时返回，为失败原因说明
    商户id shopId 是 String（32） b0fecf75457279d78ea39e030ec214f6 商户的id
    应付金额 payable 是 Double 15.00 应付金额
    优惠金额 activity 否 Dobule 2.65 优惠金额
    金额 money 是 String（20） 12.35 交易金额，即实付金额
    订单状态 status 是 Int 1/0/2 订单状态：1：支付成功；0：未支付成功；2：支付中；3：订单已关闭
    交易时间 time 是 String（32） 2016-09-28 10:20:32 交易发起时间，yyyy-MM-dd HH:mm:ss
    支付成功时间 timeEnd 否 String（20） 20180806123223 支付成功时间，格式yyyyMMddHHmmss
    聪明付订单号 cmOrderId 是 String（32） 20160922194944712595 聪明付订单号
    通道流水号 transaction_id 是 String（32） 101550066030201708281221607314 通道流水号，退款单号
    订单号 orderId 是 String（32） 20160509155415341275 商户原订单号
    手续费金额 commission 是 Double 0.23 手续费金额
    支付类型 order_type 是 String（10） weixin/alipay 支付类型，weixin：微信支付；alipay：支付宝支付
    退款状态 refundStatus 否 Int 2 退款状态，0：未退款；1：退款中；2：退款成功
    退款金额 refund_money 是 Double 50.00 退款金额
    退款时间 refund_time 否 String（20） 2016-09-29 10:20:32 最后一笔退款时间，yyyy-MM-dd HH:mm:ss
    退款详情 refund_info 否 JSONArray [{},{},{}] 退款详情，退款状态为1或2时存在

    refund_info为JSONArray类型，其中JSONObject数据字段如下：

    字段名 变量名 必填 类型 示例 描述
    聪明付退款单号 refund_order_id 是 String（32） 20180823111053456798 聪明付退款单号
    商户退款流水号 shop_refund_order_id 否 String（64） ABC123456 商户退款流水号
    原订单单号 order_id 是 String（32） 20160922194944712595 聪明付交易订单号
    退款金额 refund_money 是 Double 10.23 退款金额
    退款状态 refund_status 是 Int 0/1/2/3 退款状态，0：未退款；1：退款中；2：退款成功；3：退款失败
    返还手续费 return_commission 是 Double 0.05 返还手续费，所有手续费总和不一定与实际手续费一致，实际手续费之和以订单中额commission字段最准
    退款发起时间 refund_start_time 是 String（20） 2018-08-23 10:23:32 退款发起时间
    退款完成时间 refund_end_time 否 String（20） 2018-08-23 10:23:50 退款完成时间，退款状态为2时返回
